# Cos' week 🗓️

A modern, interactive weekly schedule builder with conflict detection and image export. Designed for students and busy individuals who want a clear, visual overview of their week.

**Live Demo:** [https://abdulqodirmir.github.io/cos-week-scheduler/](https://abdulqodirmir.github.io/cos-week-scheduler/)

## ✨ Features

- **Visual Grid Layout**: Optimized horizontal view with Days as rows and Time slots as columns.
- **Easy Event Management**: Quick add/edit/delete functionality via a clean modal interface.
- **Conflict Detection**: Real-time overlapping event detection with visual pulse alerts.
- **Theme Support**: Choose from various professional color themes for different activities.
- **PNG Export**: Export your beautiful schedule as a high-quality image to share with friends.
- **Persistent Storage**: Your schedule is automatically saved to your browser.

## 🚀 How to use

1. Open the [Live App](https://abdulqodirmir.github.io/cos-week-scheduler/).
2. Click **"Add Event"** to start building your schedule.
3. Fill in the title, pick a day, and set the start time and duration.
4. Use the **"Export PNG"** button to download your schedule.

## 🛠️ Built with

- Vanilla JavaScript
- Tailwind CSS
- Lucide Icons
- html2canvas (for image export)
